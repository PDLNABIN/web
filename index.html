About the Project

This tool helps geomatics engineers and surveyors calculate coordinates based on initial stations, distances, and internal angles. This is the assignment for the Web gis. Through this project, students were instructed to demostrate their skill of web develoment.Our task was to create a responsive website for the Coordinate calculations

Features

  • Easy input for initial coordinates, distances, and angles
  • Automatic calculation of traverse points
  • Clear display of results
  • Dynamic input fields
